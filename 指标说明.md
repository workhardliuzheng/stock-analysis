# A股投资顾问指标说明

## 📊 核心指标详解

### 1. **final_signal（最终信号）**

> **含义**: 系统综合多因子和ML预测生成的最终操作建议

| 值 | 含义 | 操作建议 |
|---|------|---------|
| `BUY` | **买入信号** | 建议建仓或加仓 |
| `SELL` | **卖出信号** | 建议减仓或清仓 |
| `HOLD` | **持有信号** | 维持现状，观察等待 |

---

### 2. **final_confidence（置信度）**

> **含义**: 系统对最终信号的有信心程度（0-100%）

| 置信度 | 信号强度 | 操作建议 |
|--------|---------|---------|
| **80%-100%** | 🔥 强烈信号 | 高仓位执行（建议80%以上） |
| **60%-80%** | ✅ 明确信号 | 中等仓位执行（建议50%-80%） |
| **40%-60%** | ⚠️ 一般信号 | 轻仓试探（建议20%-50%） |
| **0%-40%** | ❌ 弱信号 | 观望为主，小仓位或不开仓 |

**示例**: 
- `final_confidence: 82.3%` → 强烈买入，建议80%仓位
- `final_confidence: 45.6%` → 信号较弱，建议观望

---

### 3. **factor_score（多因子评分）**

> **含义**: 基于技术面的多因子综合评分（0-100分）

**计算因子**:
- ✅ **趋势因子**: MA5/10/20/50均线排列（权重40%）
- ✅ **动量因子**: RSI、MACD等 Momentum指标（权重30%）
- ✅ **成交量因子**: OBV、量能变化（权重20%）
- ✅ **估值因子**: PE/PB历史百分位（权重10%）

| 评分 | 市场状态 | 解读 |
|------|---------|------|
| **80-100分** | 🚀 过度申购 | 强势上涨，注意回调风险 |
| **60-80分** | 💪 强势上涨 | 可配置仓位，逢低加仓 |
| **40-60分** | ➡️ 震荡整理 | 观望为主，高抛低吸 |
| **20-40分** | 📉 弱势下跌 | 轻仓或空仓，等待企稳 |
| **0-20分** | 🔥 强烈卖出 | 强势空仓，等待底部信号 |

---

### 4. **ml_probability（ML上涨概率）**

> **含义**: XGBoost模型预测的上涨概率（0-100%）

**模型特点**:
- 输入: 35个技术指标特征
- 输出: 涨跌概率（回归任务）
- 窗口: 预测未来5个交易日涨跌幅

| 概率 | 预期收益 | 操作建议 |
|------|---------|---------|
| **70%-100%** | ⬆️ 强烈上涨 | 建仓，高仓位 |
| **55%-70%** | ➡️ 稳健上涨 | 加仓，中等仓位 |
| **45%-55%** | ➡️ 震荡整理 | 维持现状 |
| **30%-45%** | ⬇️ 稳健下跌 | 减仓，低仓位 |
| **0%-30%** | 🔥 强烈下跌 | 清仓，空仓观望 |

---

### 5. **trend_state（趋势状态）**

> **含义**: 当前价格趋势分类

| 状态 | 含义 | 特征 |
|------|------|------|
| `uptrend` | 📈 上升趋势 | MA5>MA10>MA20，价格在均线上方 |
| `downtrend` | 📉 下降趋势 | MA5<MA10<MA20，价格在均线下方 |
| `sideways` | ➡️ 震荡整理 | 均线缠绕，无明显方向 |

---

## 📋 完整信号解读流程

### 第一步: 看最终信号
```
BUY → 准备建仓
SELL → 准备减仓
HOLD → 维持现状
```

### 第二步: 看置信度
```
> 70% → 大仓位执行
50-70% → 中等仓位
< 50% → 轻仓或观望
```

### 第三步: 看趋势
```
uptrend → 顺势而为
downtrend → 风险较高
sideways → 高抛低吸
```

### 第四步: 综合决策
```
BUY + 高置信度 + uptrend → 全仓买入
BUY + 低置信度 + sideways → 轻仓试仓
SELL + 高置信度 + downtrend → 清仓离场
```

## 🎯 投资策略模板

###保守型策略
- 只做 `final_confidence > 75%` 的信号
- 最大仓位 50%
- SELL信号立即清仓

###平衡型策略（推荐）
- `final_confidence > 60%` 开始建仓
- 最大仓位 80%
- 分批建仓，逢低加仓
- 设置5%止损

###激进型策略
- `final_confidence > 50%` 就参与
- 最大仓位 100%
- 配合技术突破确认
- 设置3%止损

## ❓ 常见问题

### Q1: 为什么有一些指数显示"待显示"？
A: 可能是`IndexAnalyzer`还没分析完成，或者数据同步有问题。请查看日志确认分析状态。

### Q2: final_confidence和ml_probability有什么区别？
A: 
- `final_confidence`: 系统对最终信号的有信心程度（多因子+ML综合）
- `ml_probability`: ML模型预测的上涨概率（纯数据预测）

### Q3: 当多因子和ML信号冲突时怎么办？
A: 
- 多因子更看重技术面（趋势、量能）
- ML更看重历史数据模式
- 建议以`final_signal`为准（系统已综合考虑）

### Q4: 回测准确率是多少？
A: 见`BACKTEST_LOG.md`文件，通常在50-60%之间。股票市场本身具有随机性，追求长期稳定收益而非短期精准预测。
