## 🔴 高价值优化（预期收益较高）- 4项待实现

### 1. 跨指数特征 ⭐⭐⭐⭐⭐
- **现状**: 每个指数独立分析，不考虑其他指数信息
- **方案**: 
  - 用大盘指数（上证综指、沪深300）的趋势/情绪作为小盘指数的输入特征
  - 大盘/小盘相对强度（如中证500/沪深300比值的趋势）
  - 市场风格轮动信号（大盘股 vs 小盘股强弱对比）
- **预期效果**: 捕捉市场联动效应，提升小盘股预测准确性
- **涉及文件**: `analysis/ml_predictor.py`, `analysis/index_analyzer.py`
- **难度**: 中高
- **状态**: [ ] 待实现

### 2. 宏观因子 ⭐⭐⭐⭐
- **现状**: 仅使用技术面+估值数据
- **方案**: 加入宏观经济数据作为 ML 特征
  - 利率（10年期国债收益率）
  - 汇率（美元/人民币）
  - 北向资金净流入
  - 融资融券余额变化
- **预期效果**: 提升模型对市场环境的感知能力
- **涉及文件**: `analysis/ml_predictor.py`, 新建数据同步模块
- **难度**: 中高
- **状态**: [ ] 待实现

### 3. 动态权重调整 - 用Optuna学习最优多因子权重 ⭐⭐⭐⭐
- **现状**: 固定权重 (趋势30% + 动量25% + 成交量15% + 估值20% + 波动率10%)
- **方案**: 
  - 用Optuna在回测框架内搜索最优权重组合
  - 用回测结果作为优化目标
  - 可以根据市场状态动态调整权重
- **预期效果**: 提高多因子策略收益
- **涉及文件**: `analysis/multi_factor_scorer.py`
- **难度**: 中等
- **状态**: [ ] 待实现

### 4. 权重融合优化 - 混合策略自适应融合 ⭐⭐⭐⭐
- **现状**: 固定融合比例 (40%多因子 + 60%ML)
- **方案**: 
  - 用ML学习最优融合权重（meta-learner / stacking）
  - 根据市场状态动态调整多因子/ML权重
- **预期效果**: 提高混合策略收益，增强策略鲁棒性
- **涉及文件**: `analysis/signal_generator.py`
- **难度**: 中等
- **状态**: [ ] 待实现

- **预期效果**: 减少低波动指数的过度交易
- **涉及文件**: `analysis/ml_predictor.py`

- **状态**: [ ] 待实现

### 6. 市场状态识别与策略切换 ⭐⭐⭐⭐

- **方案**:

- **预期效果**: 提高策略适应性，增强收益表现
- **涉及文件**: `analysis/multi_factor_scorer.py`, `analysis/ml_predictor.py`
- **难度**: 高
- **状态**: [ ] 待实现

### 7. 模型集成优化 - 加权集成 ⭐⭐⭐
- **现状**: 简单平均集成 (XGBoost + LightGBM)
- **方案**: 
  - 用回测结果学习最优权重
  - 根据市场状态动态调整模型权重
  - 尝试更多模型（CatBoost, Neural Network）
- **预期效果**: 提高预测准确性，降低过拟合
- **涉及文件**: `analysis/ml_predictor.py`
- **难度**: 中等
- **状态**: [ ] 待实现
