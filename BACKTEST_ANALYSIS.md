## 问题分析

### ❓ 观察到的现象

1. **回测#8 (V7-2) 收益普遍高于回测#3**
   - 7/8 指数收益提升，只有中证500下降
   - 但中证500回测#3已经是+86%的高收益，回测#8为+65.87%

2. **"信号阈值自适应优化"真实影响者是谁？**
   - 回测#3 标题是"自适应阈值 + IC优化目标 + 去均值化 + Optuna调优"
   - 这是**综合优化**，不是单独的信号阈值自适应优化
   - 回测#8 标题是"特征交叉工程优化"
   - V7-2 很可能是**在V7基础上的再优化**

|------|------|----------|----------|

### 🔍 关键发现

1. **V7-2 (特征交叉优化) 确实提升了收益**
   - 回测#7 应该是 V7-2 第一次回测
   - 回测#8 是 V7-2 的仓位分配优化

2. **信号阈值自适应优化的效果无法单独分离**
   - 从回测#2 → 回测#3 是综合优化，包含多个改进
   - 从回测#7 → 回测#8 是同一版本的不同配置

3. **真正造成误解的原因**
   - TODO.md 记录"2026-03-30 (V6) - 信号阈值自适应"为已实现
   - 但 V6 和 V7 是**两个不同的优化版本**
   - 回测#3 (2026-03-27) 标题包含"自适应阈值"，实际是多个优化的组合
