### 2026-04-11 #15 - V8 市场状态识别（8指数组合回测）

**优化目标**: 解决V7-7中4个共识指标高度相关、BUY/SELL信号严重不平衡(1:2.45)、所有市场环境使用相同参数的问题

**优化方案**: 6状态市场环境识别 + 动态参数调整

**核心模块**: `analysis/market_regime_detector.py`

**6种市场状态**:
1. BULL_TREND - 牛市趋势 (趋势分>65, 波动分<55)
2. BULL_LATE - 牛市末期 (趋势分>55, 波动分>60)
3. BEAR_TREND - 熊市趋势 (趋势分<35, 波动分<55)
4. BEAR_LATE - 熊市末期 (趋势分<45, 波动分>55, 情绪分>55)
5. HIGH_VOL - 高波动 (波动分>75)
6. SIDEWAYS - 震荡市 (默认)

**3维评分体系**:
- 趋势维度: MA50斜率(35%) + 价格/MA50位置(35%) + ADX方向(30%)
- 波动维度: ATR百分位(40%) + 布林带宽度(35%) + 实现波动率比(25%)

1. `index_analyzer.py` - 在percentile计算后、multi_factor之前插入regime检测
2. `adaptive_fusion_optimizer.py` - 根据regime动态调整共识投票阈值
3. `multi_factor_scorer.py` - 根据regime动态切换5维因子权重
4. `ml_predictor.py` - 新增3个regime维度特征(trend/vol/sent score)

### 2026-04-11 #14 - V7-7 多指标共识投票（8指数组合回测）

**优化目标**: 解决2023-2025年间信号过于稀疏的问题（买入后2年无卖出信号）

**优化方案**: 在 `adaptive_fusion_optimizer.py` 的 `generate_fused_signal()` 中引入4指标共识投票

**4个独立指标**:
1. MA趋势: close > MA20 且 MA20 > MA50 -> 看涨 (+1)
2. MACD柱状图: > 0 -> 看涨 (+1)
3. RSI: > 50 -> 看涨 (+1)
4. ADX方向: +DI > -DI -> 看涨 (+1)

**非对称阈值**:
- 看涨升级 (SELL->HOLD, HOLD->BUY): 需4票全部看涨 (consensus >= 4)
- 看跌降级 (BUY->HOLD, HOLD->SELL): 需3票看跌即可 (consensus <= -3)
- 设计理念: 快速躲跌、谨慎追涨
