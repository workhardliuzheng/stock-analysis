# 2026-05-03 - 可转债数据同步模块

## 完成

### sync/sync_convertible.py + entity/convertible_bond.py
- 创建 `entity/convertible_bond.py` — ConvertibleBondBasic / ConvertibleBondDaily 实体类
- 创建 `sync/sync_convertible.py` — 数据同步模块
  - ORM 模型自动建表 (cb_basic / cb_daily)，utf8mb4 编码
  - `sync_basic()` — 同步 1125 只可转债基本信息
  - `sync_daily()` — 同步日线行情（含逐只延时 + 限流重试）
  - CLI 支持: `python sync/sync_convertible.py basic|daily|summary|all`

### 同步结果
- **基本信息**: 1125 只可转债
- **仍在交易**: 371 只
- **日线记录**: 66,682 条 (2025-05-06 ~ 2026-04-30)
- **有日线的**: 276 只
- **重复记录**: 0 组 (UNIQUE约束生效)
- **API限流**: 每只延时 0.35s，限流时自动等待 60s 重试

### 技术要点
- **表结构**: cb_basic 含 ts_code/bond_short_name/stk_code/stk_short_name/conv_price 等 17 字段；cb_daily 含 ts_code/trade_date/close/pct_chg/vol/amount 等 11 字段
- **UPSERT**: 通过 CommonMapper 的 ON DUPLICATE KEY UPDATE 实现幂等插入
- **无评级字段**: Tushare cb_basic 无 bond_rating，评级过滤需要另寻数据源
- **无 premium_ratio**: 需在策略层计算 (conv_price + stock_price)
