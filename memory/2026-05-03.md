## 下午: 双低策略模块

### analysis/cb_strategy.py 创建
- **CbDualLowStrategy** 核心类
  - `calculate_premium_ratio()` — 转股溢价率计算: `(bond_price / (100/conv_price * stock_price) - 1) * 100`
  - `calculate_dual_low()` — 双低值: `bond_price + premium_ratio`
  - `get_bonds_for_date()` — JOIN cb_daily + cb_basic + stock_daily_basic 获取数据
  - `get_recommendations()` — 返回双低排序 Top N
  - `backtest()` — 月频轮动回测

### 开发和调试过程
1. **Collation错误**: cb_basic/cb_daily 用 utf8mb4_unicode_ci, stock_daily_basic 用 utf8mb4_general_ci → 修改 cb 表 collation 统一
2. **Decimal类型错误**: MySQL Decimal 无法直接用于 Python 除法 → DataFrame 用 pd.to_numeric 转换
3. **数据不覆盖**: stock_daily_basic 最新到 2026-03-06，cb_daily 到 2026-04-30 → 推荐/回测只用有stock数据的日期
4. **回测卖出bug**: 调仓时旧持仓不在新 price_map 中无法卖出 → 一次性查询所有代码价格
5. **重合持仓bug**: 两期选中的重合代码在调仓时被重复买入 → 改用 retain + only-buy-new 逻辑
