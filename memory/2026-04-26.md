# 2026-04-26 - V16周线过滤回退

## 事件记录

### V16周线过滤存在未来数据泄漏

**问题发现**:
- 将周线过滤应用到fused_signal后，回测收益从+85.37%飙升到+279.22%
- 回撤从-28.90%降至-13.06%
- 月度胜率从50.0%升至64.5%
- 用户质疑结果异常，要求深入分析

**根因分析 (look-ahead bias)**:
- `_apply_weekly_filter()` 计算周线趋势使用当天周收盘价
- 然后将趋势合并回同一ISO周的所有交易日（包括周一到周四）
- 周一交易信号用了周五才能知道的周线趋势 → 未来数据泄漏

**验证**:
- 原始(bug): +279.22%, -13.06%
- 修复后(shift(1)): +12.18%, -42.00%
- 确认周线过滤无效，回退移除

**教训**: 时间序列数据中，用当周期数据过滤当周交易信号必然存在look-ahead bias。所有时间框架过滤必须使用前一个已完成周期的趋势。
