## 技术要点
- **价格匹配流程**: 代码精确匹配 → 代码/名称模糊匹配 → 名称子串匹配 → code子串匹配
  - 518880.SH → 直接匹配
  - 300ETF(用户持仓) → 通过名称"沪深300"(信号中) 子串匹配
- **无 emoji**: 所有标记改为 `[WARN]`、`[OK]` 等 ASCII 标记
- **不影响现有API**: `generate()` / `generate_text()` 签名不变，持仓段自动新增
